# lp-input 组件  ×

属性参数     | 可选值
-------- | ----- 
placeholder | 请输入内容(默认)
focus(是否获取焦点) | false(默认)/true
type(输入框类型) | text(默认)/password/number
name | 默认为 ''
disabled | false(默认)/true
v-model(对应value) | 默认为 ''
clearable(是否有清除功能) | false(默认)/true
showPassword(是否有查看密码功能) |false(默认)/true
size(输入框大小) | small/middle(默认)/big

事件     | 含义
-------- | ----- 
focus | 获得焦点时触发
blur | 失去焦点时触发
change | value改变时触发
